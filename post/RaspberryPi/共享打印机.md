打印机: HP 1020


## 安装打印机驱动
```bash
# 下载安装打印机驱动
wget -O foo2zjs.tar.gz http://foo2zjs.rkkda.com/foo2zjs.tar.gz

tar -xzvf foo2zjs.tar.gz

cd foo2zjs

# 编译
make

# 安装驱动
sudo make install
make install-hotplug
```

## 配置CUPS

```bash
apt install cups

service cups start

# 修改配置文件
vim /etc/cups/cupsd.conf
```
